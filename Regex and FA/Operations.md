# Operations in Regular Expressions

An empty set is represented as $\phi$, whereas an empty string is represented
as $\epsilon$

1. $a + \epsilon = a + \epsilon$

2. $a . \epsilon = a$

3. $a + \phi = a$

4. $a . \phi = \phi$

5. $\epsilon + \epsilon = \epsilon$

6. $\epsilon . \epsilon = \epsilon$

7. $\phi + \phi = \phi$

8. $\phi . \phi = \phi$

9. $\epsilon + \phi = \epsilon$

10. $\epsilon . \phi = \phi$

11. $a . (b + c) = ab + ac$

12. $a + (b . c) = a + bc$

13. $R . \phi = \phi$

14. $R + \Sigma^* = \Sigma^*$
